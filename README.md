# next-study

## 실행 방법

```bash
# 환경 설정
$ npm install

# 3000포트에 로컬 개발 환경 실행
$ npm run dev
```

## Hook 관련

- State Hook
  - Vue의 ref와 유사한 반응형 변수 관련
- Effect Hook
  - State의 변화를 감시하는 기능, Mount 될 때도 실행
